WooCommerce Payment Gateway - SCB Lite is the only WooCommerce payment gateway for Siam Commercial Bank.

== Description ==

The WooCommerce Payment Gateway - SCB Lite enables you to receive payments by credit card via the Siam Commercial Bank.
It comes with a lot of features, such as integrated fraud detection, test (sandbox) mode and clear order notes for the admins.
Simply activate it, fill out the details in the settings and start receiving funds with the lowest transaction fees.

Please note that a SCB merchant account is required in order to use this plugin.

== Installation ==

Use WordPress' Add New Plugin feature, searching "WooCommerce Payment Gateway - SCB", or download the archive and:

1. Unzip the archive on your computer  
2. Upload `woocommerce-scb-gateway-lite` directory to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to WooCommerce -> Settings -> Checkout -> SCB - Siam Commercial Bank and fill out the required fields.

== Frequently Asked Questions ==

= Do I need an account with Siam Commercial Bank in order to use this plugin? =

Yes, a SCB merchant account is required.

= Do I need any details from SCB =

Yes. In order to use this plugin you need the SCB Merchant ID and the SCB Terminal ID. Furthermore you need to activate your URL in your SCB account settings. Please contact SCB Support if you are not sure how to do this.

= What credit card details can I use in the Test mode? =

Please use either "Card number: 4907 3310 9999 9034, Last three digit (CVV2): 905, Card type: VISA (01), Card Expire Date: 10/2016" or "Card number: 5414 9635 9999 9024, Last three digit (CVV2): 307, Card type: MASTER CARD (02), Card Expire Date: 04/2016".
